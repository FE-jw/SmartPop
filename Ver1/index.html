Smart Layer Popup

1. a 태그로 적용하는 경우연결할 팝업의 ID값을 href에 설정
#POP-1 열기 
<a href="#POP-1" class="btn-pop">#POP-1</a>
2. button 태그로 적용하는 경우연결할 팝업의 ID값을 data-popup에 설정
<button type="button" class="btn-pop" data-popup="#POP-2">#POP-2</button>
특징
1. ESC key를 눌러 팝업을 닫을 수 있다.
2. 팝업을 열면 포커스가 팝업으로 이동하여 정보의 순서가 자연스럽다.
3. 닫기 버튼에서 실수로 Tab key를 눌렀을 때 다시 팝업으로 포커스가 이동한다. (키보드 이용자가 실수로 팝업을 닫지 못하는 것을 방지한다.)
4. 팝업을 닫으면 눌렀던 버튼으로 포커스가 이동하여 이어서 정보를 탐색할 수 있다.
